⏳ YOU NEED TO DO (15 minutes total):

□ STEP 1: Create Firebase Project (5 min)
  → Go to: https://console.firebase.google.com/
  → Click "Add project"
  → Enable Phone Authentication
  → Create Firestore Database
  
□ STEP 2: Get Firebase Config (2 min)
  → Project Settings → Your apps → Web
  → Copy the firebaseConfig values
  
□ STEP 3: Create .env File (3 min)
  → Create file: .env (in project root)
  → Paste your Firebase config (see SETUP_GUIDE.md)
  
□ STEP 4: Install Firebase CLI (2 min)
  → Run: npm install -g firebase-tools
  → Run: firebase login
  
□ STEP 5: Deploy Security Rules (3 min)
  → Run: firebase init
  → Run: firebase deploy --only firestore:rules

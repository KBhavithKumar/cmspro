## 📱 Responsive Breakpoints

### **Mobile (< 768px):**
```css
- Sidebar: Hidden, slide-in menu
- Navbar: Compact, no search
- Content: Full width, p-4
- Buttons: Full width
- Cards: Stacked
```

### **Tablet (768px - 1024px):**
```css
- Sidebar: Slide-in menu
- Navbar: Full features
- Content: Padded, p-6
- Buttons: Inline
- Cards: 2 columns
```

### **Desktop (≥ 1024px):**
```css
- Sidebar: Always visible, 256px
- Navbar: Full features
- Content: Max-width, centered
- Buttons: Inline
```

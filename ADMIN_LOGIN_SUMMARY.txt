═══════════════════════════════════════════════════════════════
  🔧 SETUP REQUIRED (2 Steps)
═══════════════════════════════════════════════════════════════

STEP 1: Enable Email/Password Authentication in Firebase
─────────────────────────────────────────────────────────────
1. Go to: https://console.firebase.google.com/
2. Select project: fcms2025
3. Click "Authentication" → "Sign-in method" tab
4. Find "Email/Password" → Click it
5. Toggle "Enable" → Save

STEP 2: Create Admin Accounts
─────────────────────────────────────────────────────────────
Choose ONE method:

METHOD A: Automatic (Easiest)
  Run this command:
    npm run create-admins
  
  This will create all 4 admin accounts automatically!

METHOD B: Manual (Firebase Console)
  1. Go to Firebase Console → Authentication → Users
  2. Click "Add user"
  3. Enter email: admin1@cms.local
  4. Enter password: Admin@123
  5. Click "Add user"
  6. Repeat for all 4 admins
